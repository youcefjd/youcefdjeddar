# AI Journal

My personal blog documenting my journey through artificial intelligence.

**Live site:** https://youcefjd.github.io/youcefdjeddar/

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then visit http://localhost:4000/youcefdjeddar/

## Writing Posts

Add markdown files to `_posts/` with the format:

```
YYYY-MM-DD-title-slug.md
```

With front matter:

```yaml
---
layout: post
title: "Your Post Title"
date: YYYY-MM-DD
description: "Brief description for previews"
---
```
